# 菊花厂PL实验室实习/人才引进

### **终端编程语言**

25人

岗位职责：

1、参与自研编程语言的设计；

2、参与领域语言的设计和实现；

3、参与语言跨端架构的设计和实现；

4、参与编译前端的设计和实现；

基本要求：

1、前端APP开发经验，熟悉前端APP开发的难点和问题，包括不限于Andorid、IOS、快应用、小程序、Flutter、Web（Vue、React等）；

2、熟悉前端框架的编程模型和架构，了解前端框架的演进趋势; 

3、熟练使用多种应用开发语言，包括Kotlin、Groovy、 C++ 模板、Dart、swift、JavaScript、TypeScript等；

4、了解跨语言技术，包括TS to JS、 Dart to JS、 Kotlin to JS、 Java to JS;

优选要求：

1、有DSL设计经验；

2、有编程语言编译器(Parser、Lexer、语义、codegen)实现经验；

3、有任意语言runtime实现经验，包括JVM、JS Engine、Webassembly Runtime等；

4、有编程语言的静态分析和指针分析技术相关经验；

### **语言产品规划专家**

1、产品是华为自研领域语言、通用语言以及语言IDE；

2、有工业界背景，了解多个领域的痛点，比如数通、PLC、IoT、AI，能够给领域语言DSL提出设计需求；

3、了解PL的基本理论，比如类型系统、编译原理；

4、有终端产业（手机、PC、IOT、云）和计算产业（AI、云、ARM、intel、英伟达）背景的；

5、有大局观，了解华为的鸿蒙、EMUI、HMS，以及谷歌的android、Dart、GMS；
