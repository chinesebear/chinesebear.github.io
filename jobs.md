# 菊厂编程语言实验室

### 1. 通用软件工程师

#### 岗位职责：

- 参与编程语言的项目；

#### 基本要求：

- 掌握一门或多门工业编程语言，包括Java、C++、C、Go、Kotlin、Swift、Python、Haskell、Clojure、Lisp、Rust、Php、Dart、Scala、Julia、Lua、Typescript、Ruby、JavaScript、C#等（我都快把TIOBE排名抄一遍了，应该有你懂的语言了吧）；
- 了解计算机基础知识，了解常用的基本数据结构和算法；

#### 优选要求：

- 设计过自己的语言，比如Feel语言（实验室专家自己设计的）、deeplang（小可指导设计的编程语言）
- 对某个编程语言无比热爱，像追跟网剧一样的了解该语言的每个版本。

### 2. 软件工程师（OD）

#### 岗位职责：

- 参与编程语言的项目；

#### 基本要求：

- 掌握一门或多门工业编程语言，包括C++、C、Java等；
- 了解计算机基础知识，了解常用的基本数据结构和算法；

#### 优选要求：

- 有自己的开源项目star 300+；

### 3. 编程语言设计工程师

#### 岗位职责：

1、参与自研编程语言的设计；
2、参与领域语言的设计和实现；
3、参与语言跨端架构的设计和实现；
4、参与编译前端的设计和实现；

#### 基本要求：

1、前端APP开发经验，熟悉前端APP开发的难点和问题，包括不限于Andorid、IOS、快应用、小程序、Flutter、Web（Vue、React等）；
2、熟悉前端框架的编程模型和架构，了解前端框架的演进趋势; 
3、熟练使用多种应用开发语言，包括Kotlin、Groovy、 C++ 模板、Dart、swift、JavaScript、TypeScript等；
4、了解跨语言技术，包括TS to JS、 Dart to JS、 Kotlin to JS、 Java to JS;

#### 优选要求：

1、有DSL设计经验；
2、有编程语言编译器(Parser、Lexer、语义、codegen)实现经验；
3、有任意语言runtime实现经验，包括JVM、JS Engine、Webassembly Runtime等；
4、有编程语言的静态分析和指针分析技术相关经验；
